Blink

Hello Word! da eletrônica...


Para testar as placas AL-1 e AL-2, elas possuem um Led conectado ao pino 8 do Nano.

O seguinte firmware permite ligar e desligar o led a cada segundo (1000 ms)


Blink Pin 8 
// Esta função de configuração é executada uma vez quando você pressiona reset ou liga a placa
void setup() {
    // Inicializa o pino digital 8 como uma saída
    pinMode(8, OUTPUT);
}

// A função de loop() é executada repetidamente para sempre
void loop() {
    digitalWrite(8, HIGH);   // Liga o LED (HIGH é o nível alto de tensão)
    delay(1000);             // Aguarda 1000 milissegundos que equivale a 1 segundo
    digitalWrite(8, LOW);    // Desliga o LED, tornando a tensão baixa LOW
    delay(1000);             // Aguarda 1000 milissegundos que equivale a 1 segundo
}
